 Deepfake Detection using CNN
This project detects whether a face image is real or fake using Deep Learning.

Steps performed
* Image preprocessing
* Image resizing
* Pixel normalization
* Dataset loading
* Train-test split

 Technologies used
* Python
* OpenCV
* NumPy
* TensorFlow
* Scikit-learn

Dataset
140K Real and Fake Faces Dataset
https://www.kaggle.com/datasets/xhlulu/140k-real-and-fake-faces

Project Goal
To build a machine learning model that can detect deepfake images.
